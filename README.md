# Obsidian 图像链接转换工具

一个轻量级 Python 脚本，支持将 Obsidian 中使用的图片 Wiki 链接语法 `![[图片.png]]` 转换为标准 Markdown 格式 `![](路径/图片.png)`，同时自动将图像文件名中的空格替换为中划线，提升跨平台兼容性，适用于 Typora、VS Code、GitHub 等工具。
